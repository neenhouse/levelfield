# LevelField

Anti-monopoly AI toolkit for small businesses. Detect anti-competitive AI practices, build evidence, and file complaints.

## Tech Stack
- React 19, TypeScript, Vite 8
- CSS custom properties (steel blue / justice gold theme)
- localStorage for persistence
- Cloudflare Pages deployment

## Commands
- `pnpm dev` - Dev server
- `pnpm build` - TypeScript check + Vite build
- `pnpm lint` - ESLint

## Design Identity
- Steel blue (#3B82F6) + Justice gold (#D97706)
- Dark theme, professional legal feel
- Font stack: Space Grotesk (display), Inter (body), JetBrains Mono (code)
- 8px grid spacing system
